# My Bomberman Game

## How to run this project
You should clone this repository into your machine, using the command:
```script
git clone https://github.com/Andre-Polimanti/my-bomberman.git
```
Then, create a local virtual enviroment, and activate it for this, run:
```script
python -m venv venv
source venv/bin/activate
```
Having done this, you should, finally install the dependencies listed on the requirements.txt archive contained in this project, and execute it on your machine. Do so by running:
```script
pip install -r requirements.txt
python src/main.py
```

## ControlsT
### Keyboard
+ Player 1
    - For movement, use the keyboard arrows.
    - For bombing, use the Enter key on the numeric keyboard.
+ Player 2
    - For movement, use the "WASD" arrows.
    - For bombing, use space.
## Shortcuts
+ Window
  + ESC: Close window/application
  + M: Minimize window
+ Game
  + R: Restart Game
